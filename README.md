# Work

Vue 3 + TypeScript + Vite 项目，使用 pnpm 管理依赖。

## 技术栈

- **Vue 3** — Composition API + `<script setup>`
- **TypeScript** — strict 模式
- **Vite** — 构建工具
- **pnpm** — 包管理器

## 开始

```bash
pnpm install
pnpm dev
```

## 命令

| 命令              | 说明             |
| ----------------- | ---------------- |
| `pnpm dev`        | 启动开发服务器   |
| `pnpm build`      | 类型检查 + 构建  |
| `pnpm preview`    | 预览构建产物     |
